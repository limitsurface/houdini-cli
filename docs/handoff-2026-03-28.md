## Key Live Observations

These are CLI-relevant and should be preserved in help/skill guidance later:

- `hou.Parm.valueAsData()` on component parms can return tuple-shaped data
- `hou.OpNode.parmsAsData()` can return `None`
- broad `obtain()` on Houdini object collections can fail on SWIG objects
- PowerShell inline JSON is fragile; prefer `--json -` with stdin for structured payloads
- scalar parm writes should use `parm.set(...)`, not always `setValueFromData(...)`
